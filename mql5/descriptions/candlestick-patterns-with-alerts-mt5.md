# Candlestick Patterns with Alerts MT5

**Candlestick Patterns with Alerts MT5** is a **lightweight, easy-to-configure indicator** that detects and highlights key **candlestick formations** directly on the chart. Optional alerts notify you in real time when a new pattern appears, making it ideal for **manual and discretionary traders**.

---

## What It Does

The indicator scans each completed candle and identifies selected **bullish and bearish candlestick patterns**.  
Detected formations are marked with arrows above or below the candle, providing **clear visual confirmation** and optional alerts.

---

## Supported Patterns

- Bullish Breakout  
- Bearish Breakout  
- Hammer  
- Pin Bar  
- Bullish Engulfing  
- Bearish Engulfing  
- Morning Star  
- Evening Star  

---

## Key Features

- Individual **enable / disable** control per pattern
- Optional **alerts per formation**
- Clear **arrow-based visualization**
- Custom **bullish and bearish arrow colors**
- Pattern name shown when **hovering over arrows**
- Works on **any timeframe and symbol**
- Designed for **manual trading workflows**

---

## Alert Logic

- Alerts trigger **only on a new candle**
- Alerts are sent **when the formation appears on the last completed candle**
- Each pattern can be set to:
  - Disabled  
  - Enabled (visual only)  
  - Enabled with Alert  

---

## Input Parameters

- **ShortArrowColor** – Arrow color for bearish formations  
- **LongArrowColor** – Arrow color for bullish formations  

Pattern toggles (per formation):
- Disabled  
- Enabled  
- EnabledWithAlert  
---

## How It Displays

- Bullish patterns → arrows **below** the candle  
- Bearish patterns → arrows **above** the candle  
- Hovering over an arrow shows the **pattern name**
